# Conversão de notas ( PT-BR )

Conversão de notas de 0 a 100 para valores 0 a 5 definidos dentro de intervalos:

nota <  5 = 0,5  
nota <  10 = 1,0  
nota <  15 = 1,5  
nota <  20 = 2,0  
nota <  30 = 2,5  
nota <  40 = 3,0  
nota <  50 = 3,5  
nota <  65 = 4,0  
nota <  75 = 4,5  
nota >= 75 = 5,0
